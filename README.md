### Penjelasan Menu
#### 1. Admin
#### Jika angka 1 dipilih dan login admin, maka user akan masuk ke Menu Admin. Menu untuk admin dapat melakukan CRUD (Create, Read, Update dan Delete).
#### 2. Pengguna
#### Jika angka 2 dipilih dan login pengguna maka user akan masuk ke Menu Pengguna. Menu untuk pengguna dapat melakukan Lihat Kursus dan Daftar Kursus.
#### 3. Keluar
#### Jika angka 3 dipilih, maka user akan keluar dari program.
